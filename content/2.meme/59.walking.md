## .walking(text);
Generates a Meme on Walking.

| PARAMETER   | TYPE   | DESCRIPTION                                    |
| ----------- | ------ | ---------------------------------------------- |
| text        | string | Text to Generate meme                          |

Scope: `ASYNC`
