## .bed(avatar, avatar);
Generates a Meme on Bed.

| PARAMETER   | TYPE   | DESCRIPTION                                    |
| ----------- | ------ | ---------------------------------------------- |
| avatar      | string | Avatar 1 URL to Generate meme                  |
| avatar      | string | Avatar 2 URL to Generate meme                  |

Scope: `ASYNC`
